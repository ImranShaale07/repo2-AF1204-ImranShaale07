- Python programming fundamentals
- Data handling and analysis using pandas

- Use of marimo notebooks for structured and reproducible workflows

## Project Overview
The portfolio presents a series of small data analysis tasks focused on identifying patterns, comparing variables, and communicating findings using visualisation. Across the project, I worked with structured datasets, cleaned and organised data, and selected appropriate chart types to present clear analytical conclusions.

## Analytical Focus
The project explores:
- relationships between study time and academic performance
- differences in performance across attendance categories
- trend comparisons across time-based financial data

These examples were chosen to demonstrate my ability to work with multiple data structures, compare variables, and present results in a clear and evidence-based way.

## Purpose
This project demonstrates my ability to:
- write and apply Python code to structured datasets
- analyse and interpret patterns in data
- create clear and appropriate visualisations
- communicate findings in a professional written format
- present technical work through a web-based portfolio

A key area of development in this project was moving beyond writing code and towards explaining what the results mean. Through this work, I strengthened my ability to combine technical analysis with communication, which is essential in both academic and professional data-focused environments.
