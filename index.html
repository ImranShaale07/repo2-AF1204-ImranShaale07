Project 1: Study Time Analysis

plt.tight_layout() plt.savefig("images/study_time.png") fig1 This project investigates the relationship between study time and exam performance. Using Python and pandas, I organised the dataset and examined how variation in study hours related to changes in exam score. I then used matplotlib to create a scatter plot that makes the relationship visible and easy to interpret.

The data includes student study-hour values and corresponding exam scores, allowing a direct comparison between time invested in revision and academic outcomes. The chart shows a positive relationship, with higher study hours generally associated with stronger performance.

I also extended this section by examining summary statistics, correlation, and a simple regression model in order to move beyond visual description and towards more formal interpretation.

Figure 1. Scatter plot showing a positive relationship between study hours and exam score.

Technical Journey

Throughout this module, I developed my skills progressively from foundational Python concepts towards more structured forms of data analysis, communication, and technical workflow.

In the early stages, I strengthened my understanding of variables, data types, indexing, loops, conditionals, and functions. These foundations supported later work with structured datasets, data visualisation, and interpretation.

As the module progressed, I developed practical skills in pandas for organising data, matplotlib for producing visual outputs, and marimo for structuring work in a reproducible and reactive notebook environment. I also became more confident in explaining the meaning of results rather than simply generating charts.

In later parts of the module, I extended my understanding to broader topics such as workflow automation, unstructured data, AI-supported analysis, and introductory text analysis. These topics helped me see how analytical work can move beyond static datasets towards more complex and realistic data problems.

Alongside the coding work, I also explored GitHub, GitHub Pages, HTML, and CSS in order to present my portfolio in a more professional and accessible way.
