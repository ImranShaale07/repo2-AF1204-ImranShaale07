Project 1: Study Time Analysis

This project investigates the relationship between study time and exam performance. Using Python and pandas, I organised the dataset and examined how variation in study hours related to changes in exam score. I then used matplotlib to create a scatter plot that makes the relationship visible and easy to interpret.

The data includes student study-hour values and corresponding exam scores, allowing a direct comparison between time invested in revision and academic outcomes. The chart shows a positive relationship, with higher study hours generally associated with stronger performance.

This project demonstrates my ability to work with structured data, compare two numerical variables, and communicate an analytical pattern through a suitable visual format.

Figure 1. Scatter plot showing a positive relationship between study hours and exam score.

Project 2: Attendance and Performance Analysis

This project explores the relationship between attendance rate and average academic performance. I used Python to structure the data and matplotlib to create a bar chart comparing outcomes across attendance categories.

Grouping the data by attendance level makes it possible to compare average performance across categories and identify broader patterns in the dataset. The visualisation indicates that stronger attendance is associated with higher average scores, suggesting that regular participation supports improved academic results.

This project demonstrates my ability to compare grouped data, identify category-based differences, and communicate data-driven findings clearly.

Figure 2. Bar chart comparing average academic scores across different attendance rates.

Project 3: Financial Trend Comparison

This project compares financial trends across two companies over time. Using pandas, I structured time-based data and used matplotlib to create a line chart showing how indexed performance changed across a six-month period.

Presenting the two series on the same chart makes it possible to compare overall direction, relative growth, and consistency through time. The visualisation shows that both companies experienced upward movement, although one demonstrated stronger and more sustained growth over the period.

This project demonstrates my ability to work with time-series style data, compare multiple trends within one visualisation, and interpret differences in performance.

Figure 3. Line chart comparing financial trends across two companies over time.
