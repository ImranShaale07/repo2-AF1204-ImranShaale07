Project 1: Study Time Analysis

This project investigates the relationship between study time and exam performance. I structured the dataset using Python and pandas, then used matplotlib to create a visualisation showing how exam scores change with study hours.

The chart suggests a clear positive relationship between time spent studying and academic performance. This demonstrates my ability to structure data, analyse relationships between variables, and communicate findings clearly through visualisation.

Figure 1. Scatter plot showing a positive relationship between study hours and exam score.

Project 2: Attendance and Performance Analysis

This project explores the relationship between attendance rate and average academic performance. Using Python and matplotlib, I created a bar chart to compare average score outcomes across different levels of attendance.

The visualisation suggests that stronger attendance is associated with better academic results. This project demonstrates my ability to compare grouped data, identify patterns, and communicate data-driven insights clearly.

Figure 2. Bar chart comparing average academic scores across different attendance rates.

Project 3: Financial Trend Comparison

This project compares financial performance trends across two companies over time. I used Python and pandas to structure the data, and matplotlib to create a line chart showing changes in indexed stock-price-style performance across a six-month period.

The chart shows upward movement for both companies, with one company demonstrating stronger growth over the period. This project demonstrates my ability to work with time-series style data, compare trends across variables, and present financial information visually.

Figure 3. Line chart comparing financial trends across two companies over time.
