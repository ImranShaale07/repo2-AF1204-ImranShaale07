Project 1: Study Time Analysis

This project analyses the relationship between study hours and exam performance. Using Python and matplotlib, I created a visualisation showing a positive trend between study time and academic results.

This demonstrates my ability to structure a dataset, explore relationships between variables, and communicate findings in a clear visual format.

Project 2: Attendance and Performance Analysis

This project explores the relationship between attendance rate and average academic performance. Using Python and matplotlib, I created a bar chart to compare score outcomes across different levels of attendance.

The visualisation suggests that stronger attendance is linked to better average results. This demonstrates my ability to compare grouped data and communicate patterns clearly.
